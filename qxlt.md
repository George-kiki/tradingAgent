# 情绪流龙头战法全景实战 Skill

> 本技能文档提取自《情绪流龙头战法》全书核心。
> 战法总揽“人和、地利、天时”三维共振原理。
> 专为超短线龙头交易者设计。
> 内容细化至盘面数据指标判定、K线形态辨认、分时图精解及实战仓位模型。

---

## 第一部分：总纲与核心哲学

### 1.1 战法三大基本定律
- **物竞天择，适者生存**：
  龙头股并不是自然生成的。
  它是有限资金在“选美博弈”中自然淘汰出的市场最强者。
  资金的投票决定了谁是龙。
- **强者恒强，弱者恒弱**：
  在资金的马太效应下，资金永远追逐赚钱效应最强的领头羊。
  跟风盘和中位股随时面临被抛弃的命运。
- **物极必反，否极泰来**：
  股市情绪在“恐慌”与“贪婪”中周期轮回。
  大跌之后孕育大机会。
  大涨之后潜伏大危机。

### 1.2 战法核心操作法则
- **操作铁律**：
  进出加减，步步不离龙头。
  见招拆招，招招不离题材。
- **资金原则**：
  龙头战法绝不是无脑上高度板。
  而是站在“高胜算、高赔率”的节点出击。
  机会往往只有 1.5 次，错过最佳战机坚决放弃。
- **哲学基石**：
  交易你所见，而非你所想。
  主观预判必须通过盘面走势验证。
  不与市场争辩，见势不对必须勒马回头。

## 第二部分：情报搜集与题材判定（人和）

题材是股市的第一生产力，所有的短线龙头皆由大级别题材驱动。

### 2.1 题材级别判定四大维度
- **新颖度**：
  这是全新的题材，还是炒过多次的冷饭？
  资本市场永远炒新不炒旧。
  首次曝光的题材往往具有最大的爆发力。
- **政策级别**：
  是否涉及国家高层会议定调？
  是否属于国家级重大战略规划（如一带一路、科创板等）？
- **想象空间**：
  该题材的市场容量是否足够大？
  能否容纳海量资金（数以百亿计）炒作？
  是否具有持续发酵的消息预期？
- **共鸣度**：
  该题材是否通俗易懂？
  是否容易引发普通散户与各大游资席位的广泛共鸣？

### 2.2 情报搜集日常清单
- **晚间必读**：
  新闻联播（洞察国家政策导向与最高层动态）。
  财联社/选股宝（7*24小时资讯与个股异动公告盘点）。
- **盘前必读**：
  四大证券报摘要。
  外围市场动向（美股走势、A50期指表现、汇率变动）。
- **重大信息溯源**：
  对于重磅消息，必须辨别真伪。
  需追溯国务院办公厅网站、证监会发布会、人民银行官网。
  或查阅《人民日报》等权威官媒社论。

## 第三部分：看盘与沙盘推演机制

会看盘是短线交易的地基，盘面的蛛丝马迹是验证题材真伪的唯一标准。

### 3.1 盘中四大看盘原则
- **观大局（识势）**：
  每天开盘首看大盘环境、涨跌停家数对比，判定大局风险。
  **核心指标**：盯紧**昨日涨停个股的今日表现**。
  若昨日涨停今日普遍无溢价、大幅低开，说明短线接力情绪极差，风险来临。
  若溢价良好，代表接力情绪旺盛。
- **察联动（辨流）**：
  观察板块内部联动：前排涨停是否能带动后排小弟？
  后排跟风是坚挺还是掉队？
  观察板块间资金争夺：老题材补跌资金是否流入新题材？
  主流题材是否发生切换？
- **抓重点（擒王）**：
  面对几千只股票必须剔除无效杂音。
  只看**连板高度**、**市场总龙头**、**主流板块先锋**。
  重点关注当日的“封板率”（低于70%坚决谨慎）与“涨跌家数比”。
- **重细节（决断）**：
  细节决定生死。
  重点观察龙头股竞价表现及**分时图承接力**。
  盘中下杀是否有资金迅速托底？
  是毫无抵抗的瀑布式杀跌还是量能健康的抵抗式回调？

### 3.2 盘后沙盘推演三步法
1. **盘面数据回顾**：
   记录涨停/跌停家数、昨日涨停指数、连板高度等硬核数据。
   通过数据体察情绪温度。
2. **龙虎榜逻辑分析**：
   查看龙虎榜不是为了次日无脑跟买。
   而是研究游资操盘逻辑。
   分析大资金是锁仓、砸盘还是波段加仓？
   评估题材被主流资金的认可度。
3. **制定明日交易计划**：
   资金往哪儿去？推演老弱病残退潮后，新生题材的接力方向。
   若明天龙头高开/低开该如何应对？预先设定触发条件。
   盘中像机器人一样执行（拒绝临时起意的冲动交易）。

## 第四部分：短线情绪与周期判定（天时）

指数情绪与短线情绪相辅相成。短线情绪永远在冰点与高潮之间周期性往复。

### 4.1 四大情绪周期与操作策略
1. **情绪冰点/低迷期（秋冬交替）**：
   **盘面特征**：跌停家数激增，涨停家数极少（不足20家）。
   连板空间被极限压制。
   打板被爆头率极高。
   **操作策略**：若是“首冰”，坚决空仓防守。
   若是连续冰点后的“共振冰点（指数与短线情绪双杀）”。
   则应极度敏锐，准备试错大级别新题材。
2. **情绪复苏期（初春）**：
   **盘面特征**：新题材爆发，首板、二板开始增多。
   跌停家数锐减，亏钱效应阻断。
   昨日涨停指数开始稳步回升。
   **操作策略**：敢于上仓位。
   出击新题材的领涨核心（试错龙头）。
   此时盈亏比处于最高点。
3. **情绪亢奋/高潮期（盛夏）**：
   **盘面特征**：涨停潮出现，连板高度突破前高。
   封板率超高。
   资金形成强烈的一致性看多。
   **操作策略**：享受泡沫，持股待涨。
   绝对禁止开启新的接力链条。
   “生于分歧，死于一致”。
   随时准备在情绪最亢奋、高潮顶点时兑现出局。
4. **情绪退潮期（深秋）**：
   **盘面特征**：高标龙头轰然倒塌（断板或跌停）。
   引发中位股大面积恐慌杀跌。
   亏钱效应急剧放大。
   **操作策略**：无条件清仓。
   不接飞刀，不做补涨，远离退潮漩涡。

### 4.2 情绪极值的 MACD 辅助预判
- **指数顶背离**：
  当大盘指数创出新高，但 MACD 红柱不再创新高甚至走低时。
  往往预示指数见顶。
  若此时短线情绪正处于高潮期。
  需高度警惕“指数退潮 + 情绪退潮”带来的双杀屠宰场。

## 第五部分：主力运作痕迹识别（地利）

“地利”解决的是位置与主力蓄势的问题。没有好的 K 线结构支撑，单纯靠情绪容易炸板。

### 5.1 化解“阴区”与预留“河宽”
- **阴区（套牢盘的大本营）**：
  包括密集成交带、放量大阴柱、向下跳空缺口。
  这是多头最大的阻力，不宜逆势强攻。
- **河宽（上行空间）**：
  当前股价到上方最近“阴区”的空间距离。
  河宽越广，主力拉升的阻力越小。
- **时间化解**：
  主力用漫长的横盘震荡迫使阴区套牢盘割肉。
  时间是最好的多空转换器。
- **N字结构心理战**：
  主力通过反复上冲回调（N字），不断消耗套牢盘。
  运用“诱、吓、迷、磨”四大绝招。
  对付散户的“贪、恐、疑、躁”。

### 5.2 核心起涨形态（寻找“龙行猫步”）
- **特征1：阳高阴低，阳多阴少**。
  在底部吸筹区间，阳线放量，阴线缩量。
  买方力量占据绝对主导。
- **特征2：K线紧凑优美**。
  开盘价与收盘价接近（紧盘）。
  拒绝杂乱无章的大开大合，说明主力控盘度极高。
- **特征3：波段逆势**。
  大盘暴跌期间，个股却横盘震荡甚至逆势缓涨（或连续收出十字星）。
  这是最典型的主力护盘与逆势建仓动作。
- **挖坑逆转**：
  经历一段急跌后，形成对称起涨的“坑底”结构。
  坑底右侧缩量企稳。
  一旦放量起涨，就是极佳的波段反转介入点。

## 第六部分：龙头股筛选标准（检查清单）

当出现题材爆发时，对备选个股进行逐一核对。
满足以下“是”越多的个股，越值得重仓：

- [ ] **身位第一**：该股是否在整个板块内第一时间封死涨停，占据绝对领跑身位？（是/否）
- [ ] **板块共振**：该股涨停后，是否像领头羊一样，带动了至少 3 只以上的同概念小弟涨停？（是/否）
- [ ] **题材大新**：背后的逻辑是否足够宏大，是否是首次出现的新题材（而非老题材反抽）？（是/否）
- [ ] **抗跌逆势**：在大盘低迷或指数杀跌时，该股是否表现出明显的承接力或逆势抗跌？（是/否）
- [ ] **地利优势**：起涨前是否具备“龙行猫步”量价异动或“迎龙玉带”（上方无明显套牢盘阴区）？（是/否）
- [ ] **筹码轻盈**：绝对价格是否偏低（多在15元以下），流通市值是否较小（多在20亿左右）？（是/否）
- [ ] **游资偏好**：股性是否活跃，近期是否经常出现连板或首板，无长期织布机行情？（是/否）

## 第七部分：分时图精华与细节判定

分时图是主力真实意图的实时显微镜。

### 7.1 强势分时特征
- **开盘秒板**：
  高开后不磨叽，直线拉升封板，不给任何人思考时间。
  代表资金极其一致，做多意愿坚决。
- **台阶式拉升**：
  分时图呈阶梯状上涨，每次回落低点不断抬高。
  量能配合健康（涨时放量，跌时缩量）。
- **深水捞月（地天板雏形）**：
  开盘被核按至深水区，但随后有源源不断的大单买入。
  分时线沿着均价线稳步攀升，代表承接极强。

### 7.2 弱势分时特征（必须警惕）
- **心电图式横盘**：
  分时图上下剧烈波动且毫无规律，量能萎缩。
  代表资金全在观望，主力已放弃控盘。
- **瀑布式杀跌**：
  毫无预兆地垂直下杀，中途没有任何抵抗平台。
  这是典型的主力不计成本的砸盘出货动作。
- **钓鱼线拉升**：
  盘中突然直线拉高，随后缓慢滑落，呈抛物线状。
  这是主力为了吸引跟风盘而做的诱多出货手法。

## 第八部分：实战买卖点执行手册

核心纪律：所有的买入基于逻辑发酵与形态确认，所有的卖出基于逻辑衰竭与情绪退潮。

### 8.1 买点战法
严格遵循 **“机会 1.5 次，错过最好放弃”** 的资金安全法则。

| 交易节点 | 盘面特征与触发条件 | 执行规则（当...时，应...） |
| :--- | :--- | :--- |
| **首次上车<br>(1.0次机会)** | **核心发酵日**：情绪从共振冰点回暖，大级别题材爆发，龙头股率先拉板。 | **当**该股确立为题材领涨王（板块内迅速出现≥3只助攻），且大盘风险可控**时，应**当天第一时间打板或扫板进场。 |
| **次日接力<br>(0.5次机会)** | **弱转强确认**：首日错失最佳买点，但次日龙头股高开（符合或超预期），前排稳固。 | **如果**次日龙头开盘换手健康且迅速承接上冲，**则**应趁早盘择机买入。**(注：如对次日接力环境把握不足，必须降低仓位)** |
| **N字反转<br>(加分买点)** | **洗盘结束**：首波拉升后缩量回调，经历“挖坑逆转”形态企稳。 | **如果**出现完美的N字起涨形态叠加新消息刺激，再次放量**则**视为高胜率、高赔率的第二波入场加仓点。 |

### 8.2 卖点战法
卖出不需要理由，只要不符合预期就卖出。宁可卖飞，绝不被套。

| 交易节点 | 盘面特征与触发条件 | 执行规则（当...时，应...） |
| :--- | :--- | :--- |
| **板块衰竭** | **后排掉队**：板块内连板梯队断层，跟风小弟大面积低开、翻绿或炸板。 | **当**板块整体赚钱效应枯竭，只剩龙头苦苦支撑**时，应**做好随时止盈出局的准备（独木难支）。 |
| **龙头现颓势** | **承接断裂**：龙头股竞价大幅低开且早盘无力翻红。 | **当**龙头分时图呈现明显颓势（盘中出现毫无抵抗的“瀑布式”砸盘且无资金托底）**时，应**挥剑斩仓，切忌幻想。 |
| **情绪见顶** | **高位一致后分歧**：连续高潮后，封板率骤降，昨日强势股开盘大跌。 | **如果**市场从极端一致转为严重分歧，且龙头股烂板频发（甚至大盘呈现顶背离），**则**应趁冲高果断兑现，离场空仓。 |

## 第九部分：实战案例全息推演范式

### 9.1 “大港股份”战例拆解（突发利好博弈）
- **题材缘起**：
  中美高科技摩擦升级，资金预期“自主可控/芯片”概念将大爆发。
- **初露端倪**：
  大港股份首板（虽一度被误归类为科创板跟风）。
  但深挖资料实为“集成电路测试”核心标的。
- **地利验证**：
  盘后龙虎榜显示一线游资大买。
  次日开盘无惧指数大跌回调，逆势封板。
- **板块共振**：
  随着龙头封死，次日芯片板块出现大面积涨停潮（板块效应确立）。
- **执行动作**：
  确认其为身位最高、题材最硬的龙头。
  次日直接排一字板或高开扫板进场，锁定主升浪。

### 9.2 “金力永磁”战例拆解（稀土反制）
- **题材逻辑**：
  官方媒体强硬发声“勿谓言之不预”。
  稀土概念成为最高层核心反制武器。
- **异动特征**：
  大盘大跌时该股逆势拉升，呈现标准“龙行猫步”量价护盘特征。
- **带动效应**：
  随着金力永磁拉升，英洛华、三川智慧等跟风股分时直线跟随上攻。
  典型的资金倒三角传导模式。
- **执行动作**：
  一旦确立领跑身位与带动性。
  在板块高潮爆发的第一日果断打板，享受随后的连板溢价。

## 第十部分：避坑指南与仓位模型

### 10.1 常见亏损模式拦截
1. **规避“孤勇者（一日游）”**：
   任何题材，无论逻辑听上去多么宏大。
   只要**涨停家数不足3只**（无板块效应），均视为无效题材。
   市场不认可，孤军深入必定折戟。
2. **警惕“缩量一致”的温柔一刀**：
   当题材龙头被巨单顶一字板，但后排跟风股却全线低开且毫无量能支撑时。
   这说明场外接力资金已断裂。
   前排的一字板极易在盘中遭遇爆头砸成“天地板”。
   若在场内，必须第一时间砸盘撤退，不能抱有侥幸。
3. **远离“过气老题材”反抽陷阱**：
   资金永远喜新厌旧。
   当一个题材走完完整的高潮与退潮周期后。
   第二次的消息刺激通常只会引发一日游。
   坚决规避高位老题材，只做主升浪的新题材。
4. **识破“放量震荡”陷阱**：
   在股价上方存在巨大套牢阴区时。
   个股出现放量震荡往往是解套盘抛压或主力拉高出货。
   切忌将其误认为“蓄势突破”。逆势仰攻，上涨即是穷途末路。
5. **规避“逆势接力交易”**：
   在情绪明显退潮、昨日涨停大面积核按钮的背景下。
   强行去做高标接力，无异于飞蛾扑火。顺大势，逆小势。

### 10.2 仓位管理模型
- **首冰试错仓（1成）**：
  情绪冰点末期，用极小仓位去感受盘面水温。
- **复苏跟随仓（3-5成）**：
  情绪回暖，新龙头确立首板/二板时，果断跟进半仓。
- **高潮主升仓（满仓）**：
  题材确立为主流，龙头连板，板块全面爆发时，重仓享受溢价。
- **退潮清零仓（空仓）**：
  高位龙头炸板，赚钱效应衰竭时，一刀切全部清仓防守。

## 第十一部分：股市天经与终极心法

技术（剑法）是手段，心法（执行力与情绪控制）是核心。短线交易不仅是对抗市场，更是对抗人性。

1. **先胜而后求战**：
   没有建立交易系统前，所有的交易都等同于盲目赌博。
   不战则已，战必站在概率偏向我方的一边。
   即必须达成（周期、形态、题材）的三维共振。
2. **入局有气度，出局有速度**：
   买前宜慢，千挑万选；卖时宜快，手起刀落。
   高手买在刚好（逻辑确立的第一时间）。
   次高卖在少赚（绝不贪婪最后一铜板的利润）。
   新手往往死于幻想。
3. **交易你所见，非你所想**：
   市场永远是对的。
   如果预判的龙头没有走出预期，或者开盘遭遇核按钮。
   必须马上纠错。
   拒绝自我催眠，绝不扛单死守。
4. **控制情绪，像机器一样执行**：
   人的贪（追涨）、恐（斩仓在最低点）、疑（错失良机）、躁（频繁交易）是交易的大敌。
   盘前推演定计划，盘中只负责无脑触发。
   把大脑留给复盘，把手脚留给纪律。
5. **善守者战无不胜，能舍者天地不弃**：
   放过那些不确定、级别小、或者形态别扭的所谓“机会”。
   只做确定性最高的那 1.5 次交易。
   空仓，是短线交易系统中最强大、最难修炼的一种策略。
